# Bellandur Lake Rejuvenation Tracker

Comparing what the IISc rejuvenation blueprint proposes against what Sentinel-2 satellite data actually shows about Bellandur Lake rejuvenation.

## Live Demo

- GitHub Pages: [https://shilpi1958.github.io/bellandur-lake-rejuvenation-tracker](https://shilpi1958.github.io/bellandur-lake-rejuvenation-tracker)

## Tech Stack

- React (Create React App)
- JavaScript (JSX)

## Run Locally

### Prerequisites

- Node.js 18+ recommended
- npm

### Setup

```bash
npm install
```

### Start Development Server

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000).

## Build for Production

```bash
npm run build
```

The production bundle is generated in `build/`.

## Deploy to GitHub Pages

This project is configured with:

- `homepage` in `package.json`
- `gh-pages` as a dev dependency
- deploy scripts:
  - `predeploy`: `npm run build`
  - `deploy`: `gh-pages -d build`

To deploy:

```bash
npm run deploy
```
